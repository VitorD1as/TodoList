# Todo List (Terminal - Java)

## 📌 Descrição

Aplicação simples de lista de tarefas (Todo List) executada no terminal, desenvolvida em Java. O projeto permite gerenciar tarefas com operações básicas como adicionar, listar, concluir e remover.

---

##  Funcionalidades

* Adicionar uma nova tarefa
* Listar todas as tarefas
* Marcar tarefa como concluída
* Remover tarefa por ID
